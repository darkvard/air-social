# Air Social

## Makefile Commands

| Command | Description |
|--------|-------------|
| `make up` | Start app + db + redis |
| `make down` | Stop all services |
| `make restart` | Restart the whole stack |
| `make rebuild` | Rebuild all images (no cache) |
| `make ps` | Show running services |
| `make logs` | Tail app logs |
| `make sh-app` | Shell into app container |
| `make sh-db` | Shell into Postgres |
| `make sh-redis` | Shell into Redis |
| `make air-build` | Build binary for Air reload |
